External links are links that point to a different website. For example, the link to the MDN documentation for the anchor element is an external link. External links are created using the anchor element, just like internal links. The only difference is that the href attribute points to a different page.

The anchor element can be used to create relative and absolute links. A relative link is a link that points to a page relative to the current page. For example, if you are on the page https://example.com/lessons/lesson-1.html , then a link to lesson-2.html would be a relative link. If you are on the page https://example.com/lessons/lesson-1.html, then a link to https://example.com/lessons/lesson-2.html would be an absolute link.

Relative links can also use the ../ syntax (dot-dot-slash) to point to a page relative to the current page, up one level of the directory tree. For example, if you are on the page https://example.com/lessons/lesson-1.html, then a link to ../test.html (two dot-dot-slashes) would be a relative link that points to https://example.com/test.html .

You can force a link to open in a new tab (or window) by adding the target="_blank" attribute to the anchor element. For example:

<a href="https://example.com" target="_blank">Click here to open example.com in a new tab</a>
