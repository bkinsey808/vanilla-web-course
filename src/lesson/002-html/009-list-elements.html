HTML has six heading elements, <h1> through <h6>. The <h1> element is the most important heading element, and there should be at most one per document. It is used for the main heading of the document, and if there is a <main> element in the document, the h1 element should be a heading for its content.

The subsequent headings are use for subheadings. A subheading must never skip a heading level. For example, if you have an <h2> element, you must have an <h1> element somewhere before it. If you have an <h3> element, you must have an <h2> element somewhere before it, and so on. By following these rules, you:

  • help search engines understand the structure of your page. This is important for SEO (search engine optimization)
  • help screen readers understand the structure of your page. This is important for accessibility.
