Warning: Non-optimized images can slow down your website. Optimizing images is beyond the scope of this course, and modern frameworks such as Next.js have some built-in image optimization. For a vanilla HTML/JS/CSS website, the basic recommendation is to keep your image files as small as possible.

The <img> element is used to embed an image in an HTML document. It is a self-closing element, meaning it does not have a closing tag. It has two required attributes:

  • src - the URL of the image
  • alt - a description of the image
