Hyperlinks are what makes hypertext hyper. On this page we will learn about the anchor element and how to use it to create internal links. The anchor element has an href attribute that specifies the URL of the page you want to link. Additionally, the anchor element can be used to create internal links. Internal links are links that point to a specific location on the current page.

To create an internal link, you need to create an anchor element with a name attribute. Then you build a link to this named anchor element by creating another anchor element with an href attribute. For example:

<a href="#bottom-of-doc">Click here to jump to bottom of document</a>

and then put the named anchor somewhere in your document:

<a name="bottom-of-doc"></a>

But you can see it in the Elements tab of your browser's developer tools. Note the URL has changed in your browser's address bar. It now has a # to mark the anchor. This is also called a fragment identifier.
