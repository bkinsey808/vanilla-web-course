HTML: HyperText Markup Language

HTML is the structure of a web page and can be used to add:

  • structure to text, such as headings, paragraphs, and lists.
  • meaning to text, such as emphasis or importance.
  • structure to a web page, such as navigation, sidebars, and footers.
  • media content such as images, videos, and sound
  • interactive elements, such as forms, buttons, and hyperlinks.

CSS: Cascading Style Sheets

CSS is the style of a web page and can be used to add:

  • colors, fonts, and spacing
  • animations and transitions
  • responsive design
  • and much more

JS: JavaScript

JS defines the behavior of a web page and can be used to add:

  • interactivity
  • dynamic content
  • communication with servers
  • and much more
