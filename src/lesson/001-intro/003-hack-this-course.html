This course is intended to be fully interactive. If you are at all able to set up your own coding environment, the fastest way to learn this material is to actually get a copy of this project (clone the repo), install git and npm on your operating system, and run the following commands in a terminal to get set up: 

              git clone https://github.com/bkinsey808/vanilla-web-course
              cd vanilla-web-course
              npm i -g pnpm 
              pnpm i

Then you can run the project with the command:

              pnpm start

If you are looking for a code editor recommendation, I recommend Visual Studio Code. However, the reality is many students do not have access to a computer that they can install software on. If that is the case for you, you can use an online code editor called Stackblitz. You can use it right in your browser: no need to install anything. More detailed instructions how to get set up with Stackblitz are coming. For now the important thing to know is that Stackblitz supports two code editors, the "Classic" version and the new one called Codeflow. I recommend using Codeflow if it works for you, and fall back on Classic if it doesn't. What's amazing about Codeflow is the interface is about as close to Visual Studio Code as possible in a web browser.
